# M-PESA Simulator (C++ Version)

A simple **console-based M-PESA simulation** that allows users to interact with:

- Send Money  
- Withdraw Cash  
- Buy Airtime  
- Loans & Savings  
- Lipa na M-PESA  
- My Account  

The simulation stops at **PIN entry**; no real transactions occur.

## Language Used

- **C++** – Menu-driven console application

## Purpose

- Practice **C++ programming** and **console I/O**
